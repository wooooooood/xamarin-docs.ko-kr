이 오류에서에서 볼 수 있는 Visual Studio의 오류 패드 Mac 용 또는; Visual Studio의 빌드 출력 창 Xamarin.Forms.Maps를 사용 하 여 Android 프로젝트입니다. Xamarin.Android 프로젝트에 대 한 Java 힙 크기를 늘려 가장 일반적으로 확인 됩니다.
